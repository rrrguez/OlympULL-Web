# OlympULL Web

## What is OlympULL Web?
OlympULL Web is a web application useful to manage the Computational Thinking Olympiad, a contest celebrated at University of La Laguna every year to conmemorate the Computation Studies Day.

While OlympULL Web was developed with this contest in mind, the application can be used to manage other contests in general.
